About EEG Intelligence

Research prototype for AI-assisted ADHD detection from EEG signals.

Project Overview

EEG-FM-Bench is a benchmarking framework that compares modern deep learning architectures for EEG-based ADHD classification.

The dashboard allows users to upload EEG recordings and view model predictions, confidence scores, signal visualization and saved analysis history.

Important Limitations

This application is intended for research and educational purposes only. It is not a medical diagnostic tool.

Clinical warning
ADHD diagnosis must always be made by a qualified clinician.
